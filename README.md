This is a GitHub page for the manuscript entitled: \
**"Designing for spatial Thermal Autonomy and active occupant adaption: Re-thinking the role of thermal comfort evaluation in early planning stages"** \
submitted to *Building and Environment*.

Authors: Kramer, T., Schiavon, S., Garcia-Hansen, V., Nik, V. M. (2023).


At this stage, the repository contains a the implementation of the proposed framework in a [Jupyter Notebook](/workflow/workflow.ipynb) and preliminary version of the *comfortSIM* Python [library](/lib/comfortSIM) intended for open access.

<img src="./img/climateSIM-logo.png" width="300" height="325">

With *comfortSIM* we aim to develop a module that acts as the interface between two-dimensional building simulation and common Data Science and Machine Learning libraries.

We are still in the process of streamlining the workflow notebook and converting the raw code written for the workflow into a flexible library. In a case of questions, feel free to reach out to me here on GitHub.
